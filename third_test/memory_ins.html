Memory Test Instructions

Step 1: Observe the Sequence

You will be shown a sequence of random images for 10 seconds. Focus and try to memorize the order carefully.

Step 2: Recall the Pattern

After the sequence disappears, you will be asked to recall the pattern by selecting the images in the correct order.

Step 3: Complete All Rounds

The game consists of multiple rounds with increasing difficulty. Try to score as high as possible!

Step 4: Use Focus and Memory

Stay calm and focused to improve your memory skills with each round.
